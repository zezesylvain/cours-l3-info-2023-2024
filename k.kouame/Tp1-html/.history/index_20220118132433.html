introduction

La programmation Web est la programmation informatique qui permet d'éditer des sites Web. elle permet la création d'applications destinées à être déployées sur internet ou en intranet . Ces applications Web sont constituées de pages Web pouvant prendre des formes de pages statiques et dynamiques.

principe

Lorsqu'un visiteur demande une page Web en tapant une adresse ou URL ,le serveur Web lui renvoie le contenu de la page demandée , le plus souvent en HTML ou JAVA SCRIPT OU XML .c'est ensuite le navigateur qui interprète le contenu retourné en utilisant ou non des plugins qui interprète certains objets.
